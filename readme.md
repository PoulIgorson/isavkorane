# isavkorane

Проект слушает порт 8000

Для настройки проекта:
```bash
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
```

Для запуска проекта:
```bash
python manage.py runserver
```

Получить данные для входа:
/_/get_auth/

Войти:
/accounts/login/
